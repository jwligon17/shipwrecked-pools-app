# Sprint 00 Batch 03 Prompt Packs

This batch contains four high-detail prompt packs for the Shipwrecked Pools app repo:

- `S00-006-create-permission-matrix-doc.md`
- `S00-007-create-data-visibility-rules-doc.md`
- `S00-008-create-prompt-pack-format-template.md`
- `S00-009-create-prompt-pack-status-board.md`

## How to use

1. Add this batch to the repo.
2. Commit the prompt pack files.
3. Run one prompt pack at a time with Codex using the universal runner prompt.
4. Let Codex self-review each pack.
5. Commit each completed pack result before running the next pack.

Recommended execution order:

1. S00-006
2. S00-007
3. S00-008
4. S00-009

Do not run all four prompt packs as one Codex task.
