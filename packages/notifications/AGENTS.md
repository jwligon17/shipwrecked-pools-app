# Notifications Package Agent Rules

## Purpose
Deliver reliable operational communications while respecting preferences and privacy.

## Channel Rules
- Push notifications are primary.
- SMS/email fallback may be used when defined by product flow.

## Preference Rules
- Notification preferences must be granular.
- Deal/product notifications must be independently toggleable from essential service notifications.
- Household users should have independent preference control.

## Privacy And Audit
- Respect route/privacy model (stops-before-you + ETA, no exact live GPS exposure).
- Notification send attempts and outcomes should be logged/auditable.
