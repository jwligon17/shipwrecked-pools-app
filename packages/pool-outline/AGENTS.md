# Pool Outline Package Agent Rules

## Purpose
Implement Shipwrecked's V1-critical custom top-down pool outline as structured product data.

## Core Rules
- Treat the custom outline as a core V1 feature, not decorative UI.
- Represent outline/markers as structured data, not image-only assets.
- Use normalized coordinates for marker placement across screen sizes.

## Domain Rules
- Support service points, statuses, marker history, related photos, and quote/repair linkage.
- Support separate water bodies and connected-spa visual logic.
- Keep customer-facing marker notes separate from internal operational notes.
