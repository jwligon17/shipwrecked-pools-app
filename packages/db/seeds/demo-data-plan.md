# Demo Seed Data Plan (Placeholder)

This plan defines the fixture groups expected for Shipwrecked demo/test seeds.

## Planned Fixture Groups

- Organization
- Owner/Admin users
- Technician users
- Customer and household members
- Property and access context
- Pets and permissions
- Water bodies and service points
- Route/stop placeholders
- Service report and chemistry placeholders
- Quote/repair and master-job placeholders
- Media metadata placeholders
- Notification preference placeholders
- Chat placeholders

## Data Discipline

- Placeholder IDs only.
- Placeholder dates/times only.
- No private or real customer data.
- Explicitly mark all demo records as synthetic.
